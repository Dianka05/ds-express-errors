# Contributing

Thank you for your interest in this project.

## Pull Requests

This project does **not accept pull requests** at this time.

## Bug Reports & Suggestions

If you found a bug or have a suggestion:
- Please open an issue
- Provide a minimal reproducible example
- Include Node.js version, framework, and stack trace (if available)

## Security Issues

For security-related issues, please open a private issue or contact the maintainer directly.

## Discussions

Feature ideas and questions are welcome in GitHub Discussions.
